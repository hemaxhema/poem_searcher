# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024

### Added
- Initial release of Poem Searcher
- Tashkeel-aware Arabic poem search functionality
- Comprehensive poem database with SQLite backend
- Boolean query support (AND, OR, NOT operators)
- Advanced result sorting by relevance, date, and other criteria
- Character class term groups for precise filtering
- Settings page for managing poem sources
- Persistent user preferences and source ordering
- Open source URLs directly in system browser
- Material Design UI for Windows desktop
- Full Arabic language support with proper text rendering
- Database presets and memory optimization
- Duplicate source merging functionality
- Import tooling for adding new poem sources
- Haraka-aware editing capabilities

### Features
- 🔍 Tashkeel-aware search
- 📚 Extensive poem database
- 💾 Local SQLite database (no internet required)
- 🎯 Advanced search operators
- 📊 Flexible result sorting
- ⭐ Character class filtering
- 🔗 Source URL integration
- 💾 Automatic settings persistence
- 🎨 Clean Material Design interface
